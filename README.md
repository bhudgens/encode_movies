encode_movies
=============

If you actively record video on a phone and/or DSLR and want drop any format into a folder, walk away, and return to iTunes imported compliant videos then this script is for you.

It's main purpose is to automatically encode video files in a directory to an AppleTV 3 format and import them into iTunes

A Brief Summary:
* Check source directory for video files.
* If not mp4 format transcode to mp4 (retain DD 5.1 encodings)
* Move output or MP4 files to a destination folder
* Import to iTunes
* Purge video files in destination dir that don't exist in source dir (optional)
